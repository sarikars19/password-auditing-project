# Password Auditing & Recovery Project

## Overview

This project was completed during my Cyber Security Internship at Keltron Knowledge Centre.

The objective of this educational lab was to understand password auditing and recovery techniques in a controlled and authorized environment.

## Tools Used

- Kali Linux
- Hydra
- John the Ripper
- Linux Shell

## Learning Outcomes

- Password auditing concepts
- Linux command-line usage
- Hydra for password recovery in authorized environments
- John the Ripper for password hash and ZIP password recovery
- Ethical use of cybersecurity tools

## Ethical Notice

This project was performed only in an authorized laboratory environment for educational purposes. The tools demonstrated here should only be used with explicit permission and in accordance with ethical cybersecurity practices.
